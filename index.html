About Mapnova

Mapnova is a universal GIS mapping platform pulling live data directly from Bartholomew County's official GIS server — the same source that powers Beacon/ElevateMaps — combined with statewide Indiana parcel, assessment, and spatial data.

Live Data Sources
• Bartholomew County ArcGIS MapServer — elb.elevatemaps.io (Beacon/ElevateMaps)
• Parcels, Zoning, Drains, Roads, Sections — Bartholomew Co. Surveyor/Assessor
• Esri World Imagery — Aerial basemap
• CARTO Basemaps — Streets, Dark, Light tiles
• FEMA NFHL — Floodplain data
• USDA SSURGO — Soil survey data

Layer IDs sourced directly from the BartholomewINDynamic MapServer. GIS data represents current county records maintained by the Bartholomew County Surveyor's Office, Assessor's Office, and Columbus/Bartholomew Planning Dept.
